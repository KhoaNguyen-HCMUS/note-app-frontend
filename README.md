## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Backend API running

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd note-app-frontend
```

2. Install dependencies:
```bash
npm install
```

3. Contact me to get environment variable

4. Start the development server:
```bash
npm run dev
```
